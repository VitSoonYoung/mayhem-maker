# mayhem-maker

A free fan-made tool to create custom <strong>ARAM Mayhem Augments</strong>. Choose Silver, Gold, or Prismatic tier, pick an augment icon, write your title and description, and share via a URL or export as a PNG image.

Supported keywords are automatically highlighted with colors like
Attack Damage, Ability Power, On-hit, Critical Strike, etc...
